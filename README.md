# OSINT Tool

This project is an automation tool for Open Source Intelligence (OSINT) collection.

## Features

- WHOIS Lookup
- DNS Enumeration
- Social Media Scraping
- IP Geolocation
- Passive DNS

## Requirements

- Python 3.x
- requests
- whois
- BeautifulSoup
- dns.resolver
- shodan
- tweepy
